Что я вообще могу?

Если говорить подробнее, то вот перечень моих возможностей, если так можно выразиться:
• Python: Прочитал несколько книг, могу написать небольшой платформер, текстовые игры и т.д. Хорошо знаком с циклами, условиями (if, else, elif), классами, функциями, импортированием библиотек, списками, пременными, словарями, кортежами.
Знаю про операторы чисел (деление, умножение и т.д.), "черепашьей" графике.
Конкретный пример моего проекта - Прыг-скок. В этой игре имеется платформа, которой нужно управлять с помощью стрелочек. Есть мяч, который не должен коснуться "пола".
Так же текстовая игра "приведения". Нарисованно 3 двери. Цель - не попасть в дверь с приведениями(дверь с призраком выбирается случайно). Так же есть много аналогичных игр на этом языке.

• HTML, CSS, JavaScript: Заверстал уже не один сайт, читал как книги, так и в интернете об этих языках. Хорошо знаю HTML u JavaScript, а вот CSS только самые основы. Могу прирепить катинки, аудиофайлы, гипперссылки, кнопки к сайту. Могу поставить пароль для посещения страницы, но его можно будет увидеть, посмотрев в исходный код. Сделал 2 сайта с нуля (включая этот), и 2 заверстал (последний на этапе завершения). Если у Вас есть желание ознакомиться с кодом данного сайта, нажимай сюда
.
• Командная строка Linux: Запуск команд, поиск справки о них. Основы навигации: cd, ls, pwd, which. Переменные окружения: env, export, $. Текстовый редактор nano и базовые команды vim. Права доступа в Linux: ls -l, chmod. Потоки и перенаправления: |, >>, >.
